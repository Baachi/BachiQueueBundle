# Queue bundle

[![Build Status](https://secure.travis-ci.org/Baachi/CouchDB.png?branch=master)](http://travis-ci.org/Baachi/BachiQueueBundle)

## Install ##

Add it to your composer.json file

```
{
    "require": {
        "bachi/queue-bundle": "*"
    }
}
```

### Storages ###

 * Array
 * Filesystem
 * Redis
 * RDBMS
